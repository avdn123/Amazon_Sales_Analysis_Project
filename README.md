# Amazon_Sales_Analysis_Project 
Amazon Sales Analysis Project

Project Overview

This project analyzes Amazon sales data using SQL to uncover valuable business insights. The analysis focuses on sales performance, product categories, customer purchasing trends, and revenue patterns.

Tools Used

- SQL Server
- SQL Queries
- Excel

Project Objectives

- Analyze sales performance
- Identify top-performing products and categories
- Study revenue and profit trends
- Generate business insights using SQL queries

Files Included

- amazon_sales.csv (Dataset)
- amazon_sales_analysis.sql (SQL Queries)
- Amazon_Sales_Analysis_Reports.pdf (Project Report)

Key Skills Demonstrated

- Data Cleaning
- Data Analysis
- SQL Query Writing
- Aggregate Functions
- Group By and Having Clauses
- Business Insight Generation

Conclusion

The project demonstrates the use of SQL for real-world business data analysis and reporting. It highlights the ability to extract meaningful insights from sales data to support decision-making.
